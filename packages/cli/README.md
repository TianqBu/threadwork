# threadwork-cli

**简体中文** | [English](#english)

CLI-native 多 Agent 协作工具，给 Claude Code 用。可热插拔的角色
yaml + 持久记忆 MCP（SQLite + FTS5）+ step-level trace replay。

```bash
npm install -g threadwork-cli
threadwork init
threadwork roles list
```

完整文档与源码：https://github.com/TianqBu/threadwork

## Bin

只有一个 bin entry：`threadwork`，无别名。
