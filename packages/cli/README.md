# threadwork-cli

CLI-native multi-agent collaboration for Claude Code. Hot-pluggable role
yamls, persistent memory MCP (SQLite + FTS5), step-level trace replay.

```bash
npm install -g threadwork-cli
threadwork init
threadwork roles list
```

Full docs and source: https://github.com/REPLACE_OWNER/threadwork

## Bin

The package installs the `threadwork` binary (one bin entry, no aliases).
